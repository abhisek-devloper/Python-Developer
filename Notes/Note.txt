5. Python Advanced Features
---------------------------
-Lambdas  (one line function syntax- lambda arguments: expression)
-Iterators & Iterables
	Iterable → object that can return an iterator (e.g., list, tuple, string).
	Iterator → object with __iter__() and __next__() methods.
-map, reduce, filter
      example - # map → apply function to all elements
			nums = [1, 2, 3, 4, 5]
			squares = list(map(lambda x: x*x, nums))
			print("\nMap (Squares):", squares)
		# filter → filter elements by condition
			evens = list(filter(lambda x: x % 2 == 0, nums))
			print("Filter (Evens):", evens)

		# reduce → apply cumulative function
			sum_all = reduce(lambda a, b: a+b, nums)
			print("Reduce (Sum):", sum_all)

-Generators & Decorators
	Generators → functions that use yield to return values one at a time.They save memory compared to lists.
	Decorators → functions that take another function as input and add extra functionality.
- yield (gives a sequence of results over time, without losing the function’s state.) 
  return (return gives a single result and ends the function.) 
  
-Closures (A closure is a function that “remembers” variables from its enclosing scope, even if that scope has finished execution.)
-Context Managers
 (Used to manage resources like files, network connections, etc.)
 (Implemented using with keyword (automatically handles setup & cleanup).)

6. Python OOP (Object-Oriented Programming)
-------------------------------------------
-Classes and Objects
-Inheritance
-Polymorphism
-Encapsulation
	-Definition: Wrapping variables and methods inside a class, restricting direct access to some components.
	 Achieved using private (_ or __) variables/methods.
-Abstraction
	-Definition: Hiding implementation details and showing only essential features.
	 Achieved using abstract classes (from abc module).
-Methods & Dunder (magic) methods
	-Methods: Functions defined inside a class that operate on objects.
	-Dunder methods: Special methods with double underscores (__method__). Used for operator overloading and object customization.
